# Cutover Week Check-Ins (No Sign-Off)

> Reviewed: 2026-03-06 | Status: active | Script version: v2.2

## Use This Page

Read this first during the first 7 days after cutover. Observational only, no formal approval fields.

## Daily Cadence (5-10 minutes)

- Check `Home` run outcome
- Review `Backstock Alerts` and `Compliance Alerts`
- Capture any mismatch between expectation and output
- Route issues via runbooks if blocked/failed outcomes appear

## Day 1-7 Prompts

### Day 1

- Was workflow completion faster, slower, or unchanged?
- Any confusion in checklist/compliance interpretation?

### Day 2

- Any unknown locations queued that need mapping?
- Did recommendations align with vault reality?

### Day 3

- Any repeated flags that indicate config tuning need?
- Any false positives in compliance perception?

### Day 4

- Are staff following output order consistently?
- Any friction points in escalation path?

### Day 5

- Any quality drift in imported data?
- Any repeated run outcomes needing investigation?

### Day 6

- Are manager overrides becoming less frequent?
- Which remaining objections are still unresolved?

### Day 7

- What changed most in speed, visibility, and confidence?
- What single improvement would reduce friction next week?
